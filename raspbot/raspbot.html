Sobre el Proyecto

RaspBot: Robótica Educativa Accesible

Raspbot es un robot móvil multiusos impulsado por inteligencia artificial, diseñado para procesar imágenes y detectar objetos en tiempo real mediante una Raspberry Pi 4. Su avanzada tecnología le permite no solo desplazarse por control remoto, sino también mantener conversaciones con el usuario, adaptándose a diferentes roles según las necesidades.

Gracias a su versatilidad, Raspbot puede actuar como un asistente educativo, un robot explorador o incluso un agente de monitoreo inteligente. Su capacidad para interpretar el entorno y responder de manera autónoma lo convierte en una herramienta poderosa tanto para la educación como para la investigación en robótica e inteligencia artificial.

Conocer características

Visión Inteligente

Equipado con un sistema de visión por computadora que le permite detectar objetos, reconocer rostros y analizar su entorno en tiempo real.

Procesamiento Avanzado

Utiliza una Raspberry Pi 4 como núcleo, garantizando un rendimiento eficiente para el análisis de datos y el control de sus funciones.

Interacción Natural

Capaz de mantener conversaciones con el usuario, responder preguntas y ejecutar comandos según el rol asignado.

Pasos para Armarlo

Paso 1: Preparación

Reúne todos los componentes necesarios para construir tu Raspbot. Asegúrate de tener a mano:

  • Raspberry Pi 4
  • Chasis del robot
  • Motores DC (x2)
  • Controlador L298N
  • Sensores varios
  • Cables y conectores
  • Batería portable

Paso 2: Montaje del Chasis

Ensambla el chasis del robot siguiendo estas instrucciones:

  1. Une las piezas del chasis según el diagrama
  2. Instala los motores en sus soportes
  3. Coloca las ruedas y asegúralas
  4. Instala el soporte para la Raspberry Pi

Paso 3: Conexiones Eléctricas

Realiza todas las conexiones siguiendo el esquema eléctrico:

  1. Conecta el controlador L298N a la Raspberry Pi
  2. Conecta los motores al controlador
  3. Instala y conecta los sensores
  4. Verifica todas las conexiones antes de encender

Paso 4: Configuración del Software

Configura el software en tu Raspberry Pi:

  1. Instala Raspberry Pi OS
  2. Configura la conexión Wi-Fi
  3. Instala las bibliotecas necesarias (RPi.GPIO, etc.)
  4. Descarga el código fuente del Raspbot

Paso 5: Pruebas y Ajustes

Realiza pruebas para verificar el funcionamiento:

  1. Prueba de motores y movimiento
  2. Calibración de sensores
  3. Prueba de evitación de obstáculos
  4. Ajustes finales de rendimiento

Ejemplo de Código

from raspbot.sensors import UltrasonicSensor
from raspbot.motors import MotorController

class Raspbot:
    def __init__(self):
        self.ultrasonic = UltrasonicSensor()
        self.motor = MotorController()
    
    def avoid_obstacles(self):
        distance = self.ultrasonic.get_distance()
        if distance < 30:
            self.motor.stop()
            self.motor.turn_left()
        else:
            self.motor.forward()
