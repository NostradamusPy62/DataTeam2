Sobre el Proyecto

¿Qué es RaspBot?

RaspBot es un robot móvil inteligente desarrollado sobre una Raspberry Pi 4 y un Arduino Uno. Combina inteligencia artificial, visión computacional, sensores modulares, interacción por voz y control remoto para convertirse en una herramienta educativa, de monitoreo y experimentación. Es accesible, personalizable y pensado para estudiantes, docentes y entusiastas de la robótica e IA.

Aplicaciones y Potencial

RaspBot tiene un alto potencial en distintos campos gracias a su diseño modular y multifuncional:

  • Educación: Ideal para enseñar robótica, programación e inteligencia artificial en aulas y talleres.
  • Investigación: Plataforma versátil para pruebas de algoritmos de visión, sensores y automatización.
  • Monitoreo: Capaz de observar y reportar condiciones del entorno en tiempo real, útil en laboratorios o espacios de trabajo.
  • Experimentación DIY: Base accesible para quienes quieran desarrollar sus propios proyectos con IA.

Visión Inteligente

Equipado con un sistema de visión por computadora que le permite detectar objetos, reconocer rostros y analizar su entorno en tiempo real.

Procesamiento
Avanzado

Utiliza una Raspberry Pi 4 como núcleo, garantizando un rendimiento eficiente para el análisis de datos y el control de sus funciones.

Interacción Natural

Capaz de mantener conversaciones con el usuario, responder preguntas y ejecutar comandos según el rol asignado.

Evolución del Proyecto

Etapa 1: Conceptualización

Se define el objetivo del proyecto: crear un asistente robótico educativo con capacidades de IA. Se eligen los componentes base, como la Raspberry Pi 4 para el procesamiento y la mBot2 como plataforma de movilidad. Se plantea una arquitectura modular que permita integrar sensores, cámara y sistemas de control.

Etapa 2: Prototipo Inicial

Se desarrolla el primer modelo funcional combinando la Raspberry Pi 4 con la mBot2, utilizando conexiones básicas para el control de motores y la integración de sensores. Se prueba la movilidad, se establece la comunicación entre módulos y se empieza a trabajar con visión computacional básica.

Etapa 3: Mejora del Hardware

Se optimiza la estructura física del RaspBot, añadiendo más sensores (distancia, temperatura, humedad), una cámara de mejor calidad y mejorando el sistema de montaje. Se afina la integración entre la Raspberry Pi, la mBot2 y otros periféricos para lograr un funcionamiento más robusto.

Etapa 4: Desarrollo de Software

Se programa en Python un sistema inteligente que permite procesamiento de imagen, interacción por voz, control remoto desde una app y automatización de tareas. Se integran módulos de IA para visión computacional, comandos de voz y respuesta contextual.

Etapa 5: Implementación Educativa

RaspBot se introduce en entornos educativos para validar su aplicación como herramienta didáctica. Se ajustan las interfaces y se documenta todo el proceso para facilitar su adopción por estudiantes, docentes y makers.

Características

Modelo de RaspBot

El RaspBot es un robot móvil con inteligencia artificial montado sobre una base mBot2, lo que le otorga gran estabilidad y movilidad. Está diseñado para ser una herramienta educativa accesible y poderosa, capaz de adaptarse a distintos contextos.

  • Programación en Python - El sistema está desarrollado en Python, el lenguaje de programación más popular en IA, lo que permite que cualquier persona con conocimientos básicos pueda modificar y expandir sus funcionalidades.
  • Procesamiento sensorial - Con su cámara y micrófono, el RaspBot puede reconocer objetos, interpretar el entorno y mantener conversaciones básicas con los usuarios.
  • Movilidad Inteligente con mBot2 - Gracias a la plataforma mBot2, el robot tiene movimiento preciso, control de motores eficiente y la capacidad de desplazarse de manera autónoma o dirigida en distintos entornos.

Ejemplo de Código

from raspbot.sensors import UltrasonicSensor
from raspbot.motors import MotorController

class Raspbot:
    def __init__(self):
        self.ultrasonic = UltrasonicSensor()
        self.motor = MotorController()
    
    def avoid_obstacles(self):
        distance = self.ultrasonic.get_distance()
        if distance < 30:
            self.motor.stop()
            self.motor.turn_left()
        else:
            self.motor.forward()
